# Drone Detection — YOLOv8 + OAK-D Lite + Jetson Nano

Real-time drone detection system using a custom-trained **YOLOv8** model deployed on **OAK-D Lite** camera and running on **Jetson Nano**.

---

## How It Works

```
Custom Dataset → YOLOv8 Training → .pt → .blob conversion → OAK-D Lite → Real-time detection
```

1. Raw images collected and labeled manually using **labelImg** (YOLO format)
2. Dataset split 80/20 train/val using `prepare_dataset.py`
3. **YOLOv8n** model trained using Ultralytics
4. Model converted from `.pt` → `.blob` via [Luxonis Hub](https://hub.luxonis.com/ai) (RVC2 / Myriad-X)
5. Deployed on **OAK-D Lite** camera — inference runs on-device (VPU)
6. Live video feed with bounding boxes displayed via OpenCV

---

## Hardware

| Component | Description |
|---|---|
| OAK-D Lite | Spatial AI camera — runs YOLOv8 blob on-device (Myriad-X VPU) |

## Training Your Own Model

### 1. Label your images
```bash
pip install labelImg
labelImg
```
- Open Dir → `dataset/images_raw`
- Change Save Dir → `dataset/labels_raw`
- Format: **YOLO**

### 2. Split dataset
```bash
python prepare_dataset.py
```

### 3. Configure data.yaml
```yaml
train: dataset/images/train
val: dataset/images/val
nc: 1
names: ["drone"]
```

### 4. Train YOLOv8
```bash
pip install ultralytics
yolo task=detect mode=train model=yolov8n.pt data=data.yaml imgsz=640 epochs=50 batch=8
```

### 5. Convert to blob
Go to [https://hub.luxonis.com/ai](https://hub.luxonis.com/ai) → Convert Model → YOLO  
- Platform: **RVC2**
- IR Version: **2021.4.0**
- Shaves: **6**
- Input shape: **352 x 640**

Place the output `.blob` file in `models/`.
